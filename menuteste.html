Conhecendo um pouco o JavaScript

O que é um documento JavaScript: DOM no JavaScript representa Document Object Model( Documento do modelo de Objetos), com isso o JavaScript trabalha em páginas HTML/XML, para injetar lógica pois sem o JavaScript a página seria estática e sem inteagência e essa inteagência é chamada de DOM.

DOM é a modificação e a programação injetada na página pelo JavaScript, ele tem a capacidade de ler as tags HTML,injetar tags, excluir e modificar no documento.

Document é um comando do JavaScript, pois através dele que podemos fazer as modificações no HTML e essas possibiadades de modificações é chamada de DOM.

O JavaScript é uma linguagem que suporta estilos de programaçaõ orientada a eventos, funcionais e imertivos. Criado em 1995 para tirar a cara estatica quer os sites estavando tendo trazendo mais tecnologia para eles.
